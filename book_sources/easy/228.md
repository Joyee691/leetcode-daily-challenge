# 228. Summary Ranges

> [Leetcode link](https://leetcode.com/problems/summary-ranges)

## 题目简介

```js
/**
 * @param {number[]} nums
 * @return {string[]}
 */
```

题目给我们一个由数字组成的数组

要求我们给出数组中数字的范围

## 解题思路

直接循环数组，在循环开始的时候标记开始的下标为 start

然后循环判断连续的数字持续到哪个下标（用 end 表示）

最后比价 start 与 end，如果相等，只需要保存当前下标的元素；否则需要保存从 start 到 end 的范围

### Javascript

```javascript
/**
 * @param {number[]} nums
 * @return {string[]}
 */
var summaryRanges = function (nums) {
    const res = []

    for (let i = 0; i < nums.length; i++) {
        let start = i
        while(i + 1 < nums.length && nums[i]+1 === nums[i+1]) {
            i++
        }

        if(i !== start) {
            res.push(`${nums[start]}->${nums[i]}`)
        } else {
            res.push(`${nums[start]}`)
        }
    }
    return res
};
```

### 复杂度分析
#### 时间

O(n)

#### 空间

O(1)
