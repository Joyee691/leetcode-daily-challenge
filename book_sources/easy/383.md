# 383. Ransom Note

> [Leetcode link](https://leetcode.com/problems/ransom-note)

## 题目简介

```js
/**
 * @param {string} ransomNote
 * @param {string} magazine
 * @return {boolean}
 */
```

题目要求我们从 magazine 中拿到字符拼成 ransomNote，每个 magazine 中的字符只能使用一次

如果 magazine 中的字符可以组成 ransomNote 则返回 true，否则返回 false

## 解题思路

这道题关键的限制在于 magazine 各个字符的数量

所以我们用一个 map 来记录 magazine 各个字符的数量，然后遍历 ransomNote

当 ransomNote 某个字符 map 中没有的时候，就返回 false，否则遍历结束返回 true

### Javascript

```javascript
/**
 * @param {string} ransomNote
 * @param {string} magazine
 * @return {boolean}
 */
var canConstruct = function(ransomNote, magazine) {
    const magazineMap = new Map()

    for(const char of magazine) {
        magazineMap.set(char, (magazineMap.get(char) || 0) + 1)
    }

    for(const char of ransomNote) {
        if(!magazineMap.has(char)) {
            return false
        }
        const count = magazineMap.get(char)
        if(count <= 0) {
            return false
        } else {
            magazineMap.set(char, count - 1)
        }
    }
    return true
};
```

### 复杂度分析
#### 时间

O(n)

#### 空间

O(n)
