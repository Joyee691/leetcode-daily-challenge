# 125. Valid Palindrome

> [Leetcode link](https://leetcode.com/problems/valid-palindrome)

## 题目简介

```js
/**
 * @param {string} s
 * @return {boolean}
 */
```

题目给我们一个包含任意字符的字符串

要求我们判断经过以下操作的字符串是否符合回文要求：

1. 将大写字母全部转换为小写
2. 将除了小写字母以及数字的字符删掉

## 解题思路

分为两个步骤：

1. 按照要求转换字符串
2. 判断回文

第一步用一个 `toLowerCase` 方法与正则就可以搞定

第二部需要用到双指针往中间遍历判断

### Javascript

```javascript
/**
 * @param {string} s
 * @return {boolean}
 */
var isPalindrome = function(s) {
    s = s.toLowerCase().replace(/[^a-z0-9]/g, '');
    let left = 0
    let right = s.length - 1
    while(left < right) {
        if(s[left] !== s[right]) {
            return false
        }
        left++
        right--
    }
    return true
};
```

### 复杂度分析
#### 时间

O(n)

#### 空间

O(1)
