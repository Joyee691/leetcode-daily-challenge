# 205. Isomorphic Strings

> [Leetcode link](https://leetcode.com/problems/isomorphic-strings)

## 题目简介

```js
/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
```

题目给我们两个字符串 s 与 t

要求我们判断 s 与 t 是否为 *isomorphic*（同构）

同构代表 s 中的字符通过转换可以变成 t，转换规则是：

1. 任意字符 a 可以也只可以转换成任意字符 b（b 可以等于 a）
2. 不允许有多个字符转换为同一个字符

## 解题思路一：两次判断

我们可以用一个 map 记录 s 中的字符转换为 t 中的哪个字符

如果有 s 中的某个字符转换为了两个 t 的字符，则返回 false

用上述方法我们可以判断是否满足简介中的第一个条件

接着我们用另一个 map 记录 t 中的字符转换为 s 中的哪个字符，然后用相同的条件判断，这样可以判断是否满足第二个条件

当两个条件都满足则返回 true

### Javascript

```javascript
/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
var isIsomorphic = function (s, t) {
    const len = s.length
    const stMap = new Map()
    for (let i = 0; i < len; i++) {
        if (stMap.has(s[i])) {
            if (stMap.get(s[i]) !== t[i]) {
                return false
            }
        } else {
            stMap.set(s[i], t[i])
        }
    }

    const tsMap = new Map()
    for (let i = 0; i < len; i++) {
        if (tsMap.has(t[i])) {
            if (tsMap.get(t[i]) !== s[i]) {
                return false
            }
        } else {
            tsMap.set(t[i], s[i])
        }
    }

    return true
};
```

### 复杂度分析

#### 时间

O(n)

#### 空间

O(n)

## 解题思路二：用下标来记录映射

还有另一种思路是我们可以把 s 与 t 中的相同字符都映射到第一次出现的下标

这样一来如果遍历的两个字符映射到了不同下标，则表示两个字符串不是同构，则返回 false

### Javascript

```js
/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
var isIsomorphic = function (s, t) {
    const len = s.length
    const mapS = new Map()
    const mapT = new Map()

    for (let i = 0; i < len; i++) {
        if(!mapS.has(s[i])) {
            mapS.set(s[i], i)
        }

        if(!mapT.has(t[i])) {
            mapT.set(t[i], i)
        }

        if(mapS.get(s[i]) !== mapT.get(t[i])) {
            return false
        }
    }

    return true
};
```

### 复杂度分析

#### 时间

O(n)

#### 空间

O(n)
