# 290. Word Pattern

> [Leetcode link](https://leetcode.com/problems/word-pattern)

## 题目简介

```js
/**
 * @param {string} pattern
 * @param {string} s
 * @return {boolean}
 */
```

这题跟 [205](./205.md) 很像，区别在于需要处理字符与字符串之间的映射

pattern 是一组字符的集合，题目要求我们判断 s 是否符合 pattern 的规则：**pattern 的字符与 s 中的单词是一一对应的关系**

## 解题思路

解题思路可以参考 [205](./205.md)

也就是吧 pattern 的字符与 s 中的单词都映射到第一次出现的下标

这样一来如果遍历的两个字符映射到了不同下标，则表示不符合题目要求，返回 false

这一题还需要额外判断 pattern 以及 s 中的单词数量是否一致，如果不一致也需要返回 false

### Javascript

```javascript
/**
 * @param {string} pattern
 * @param {string} s
 * @return {boolean}
 */
var wordPattern = function (pattern, s) {
    const len = pattern.length
    const words = s.split(' ')

    if(len !== words.length) {
        return false
    }

    const patternMap = new Map()
    const wordMap = new Map()

    for (let i = 0; i < len; i++) {
        if(!patternMap.has(pattern[i])) {
            patternMap.set(pattern[i], i)
        }
        if(!wordMap.has(words[i])) {
            wordMap.set(words[i], i)
        }

        if(patternMap.get(pattern[i]) !== wordMap.get(words[i])) {
            return false
        }
    }

    return true
};
```

#### 时间

O(n)

O(n)
