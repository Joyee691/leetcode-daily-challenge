# 83. Remove Duplicates from Sorted List

> [Leetcode link](https://leetcode.com/problems/remove-duplicates-from-sorted-list)

## 题目简介

```js
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
```

题目给我们一个链表头，要求删除链表中所有重复元素

最后返回没有重复元素的链表头

## 解题思路

用双指针：快指针负责寻找重复元素，慢指针负责修改 next 指针指向

### Javascript

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var deleteDuplicates = function(head) {
    let fast = head
    let slow = head

    while(slow) {
        while(fast.next && fast.next.val === slow.val) {
            fast = fast.next
            slow.next = fast.next
        }
        slow = slow.next
    }

    return head
};
```

### 复杂度分析
#### 时间

O(n)

#### 空间

O(1)
