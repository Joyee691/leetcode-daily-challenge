# 100. Same Tree

> [Leetcode link](https://leetcode.com/problems/same-tree)

## 题目简介

```js
/**
 * @param {TreeNode} p
 * @param {TreeNode} q
 * @return {boolean}
 */
```

题目给我们两颗二叉树 p 以及 q，要求我们判断他俩是否相同

相同的条件：

1. 两个树有一样的结构
2. 在同样位置的两个元素相等

## 解题思路

这地我们可以用递归的思路来做

首先对于根节点 p 跟 q 来说，有三种可能：

1. 两个阶段都为空
2. 两个节点都不为空且节点元素相同
3. 其他情况（包括一个为空一个不为空、节点元素不同）

其中第一种可能性可以直接返回 true（代表相同）

第二种可能性需要递归遍历子节点判断

第三种情况直接返回 false（代表不相同）

### Javascript

```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} p
 * @param {TreeNode} q
 * @return {boolean}
 */
var isSameTree = function (p, q) {
    if(!p && !q) {
        return true
    }
    if (p && q && p.val === q.val) {
        return isSameTree(p.left, q.left) && isSameTree(p.right, q.right)
    }
    return false
};
```

### 复杂度分析
#### 时间

O(n)：最坏情况需要遍历所有节点

#### 空间

平均情况：O(logn) 出现在树很平衡的情况

最坏情况：O(n) 出现在只有单边有节点的树上
