# 380. Insert Delete GetRandom O(1)

> [Leetcode link](https://leetcode.com/problems/insert-delete-getrandom-o1)

## 题目简介

```js
var RandomizedSet = function() {};

/** 
 * @param {number} val
 * @return {boolean}
 */
RandomizedSet.prototype.insert = function(val) {};

/** 
 * @param {number} val
 * @return {boolean}
 */
RandomizedSet.prototype.remove = function(val) {};

/**
 * @return {number}
 */
RandomizedSet.prototype.getRandom = function() {};
```

题目要求我们实现一个 `RandomizedSet` 类，包含三个方法：

- insert：插入一个数字，如果数字已经存在则返回 false；否则返回 true
- remove：删除一个数字，如果数字不存在返回 false；否则删除后返回 true
- getRandom：从当前存在的数字中随机获取一个数字

最后复杂度要求这三个算法复杂度都是 O(1)

## 解题思路

我们可以用数组来保存数字，然后用 Map 来加快查找速度

### Javascript

```javascript

var RandomizedSet = function () {
    this.list = []
    this.map = new Map()
};

/** 
 * @param {number} val
 * @return {boolean}
 */
RandomizedSet.prototype.insert = function (val) {
    if (this.map.has(val)) {
        return false
    }

    this.list.push(val)
    this.map.set(val, this.list.length - 1)
    return true
};

/** 
 * @param {number} val
 * @return {boolean}
 */
RandomizedSet.prototype.remove = function (val) {
    if (!this.map.has(val)) {
        return false
    }

    const idx = this.map.get(val)
    const len = this.list.length
    this.list[idx] = this.list[len - 1]
    this.map.set(this.list[idx], idx)
    this.map.delete(val)
    this.list.pop()
    return true
};

/**
 * @return {number}
 */
RandomizedSet.prototype.getRandom = function () {
    const randomIdx = Math.floor(Math.random() * this.list.length)
    return this.list[randomIdx]
};
```

### 复杂度分析
#### 时间

O(1)

#### 空间

O(n)：

- list：O(n)
- Map: O(n)
