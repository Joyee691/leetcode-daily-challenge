# 122. Best Time to Buy and Sell Stock II

> [Leetcode link](https://leetcode.com/problems/best-time-to-buy-and-sell-stock-ii)

## 题目简介

```js
/**
 * @param {number[]} prices
 * @return {number}
 */
```

题目给我们一个数字数组 prices

我们可以在同一天无限买卖股票，但是不能持有超过 1 股

题目要求我们求出在最优的情况下的最大收益是多少

## 解题思路

由于我们在同一天的可以无限买卖，且可以根据后面的股价决定前一天的决策，所以我们理论上可以在第二天股价比第一天高的情况下通过在第一天买、第二天卖的策略取得收益

最大的收益就是把所有的收益加总

### Javascript

```javascript
/**
 * @param {number[]} prices
 * @return {number}
 */
var maxProfit = function (prices) {
    let profit = 0
    for (let i = 1; i < prices.length; i++) {
        if(prices[i] > prices[i-1]) {
            profit += prices[i] - prices[i-1]
        }
    }

    return profit
};
```

### 复杂度分析
#### 时间

O(n)

#### 空间

O(1)
