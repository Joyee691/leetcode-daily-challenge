# 106. Construct Binary Tree from Inorder and Postorder Traversal

> [Leetcode link](https://leetcode.com/problems/construct-binary-tree-from-inorder-and-postorder-traversal)

## 题目简介

```js
/**
 * @param {number[]} inorder
 * @param {number[]} postorder
 * @return {TreeNode}
 */
```

题目给我们一颗树经过后序遍历的数组 postorder 以及中序遍历的数组 inorder

要求我们根据这两个数组还原出树并返回

## 解题思路

本题思路跟 [105](./105.md) 基本一样，区别只在于 preorder 是从数组头开始取根节点；postorder 要从数组尾取根节点

### Javascript

```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {number[]} inorder
 * @param {number[]} postorder
 * @return {TreeNode}
 */
var buildTree = function(inorder, postorder) {
    const len = inorder.length

    const map = new Map()
    for(let i=0;i<len;i++) {
        map.set(inorder[i], i)
    }
    let postorderIdx = len-1

    const buildByRange = (start, end) => {
        if(start < end) {
            return null
        }

        const value = postorder[postorderIdx--]
        const idx = map.get(value)
        const node = new TreeNode(value)
        node.right = buildByRange(start, idx+1)
        node.left = buildByRange(idx-1, end)
        return node
    }

    return buildByRange(len-1, 0)
};
```

### 复杂度分析
#### 时间

O(n)：每个节点都需要被创建一次

#### 空间

O(n)

分解的话是 O(n) + O(h) 前者代表 map 的空间，后者代表递归的空间占用
