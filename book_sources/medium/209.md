# 209. Minimum Size Subarray Sum

> [Leetcode link](https://leetcode.com/problems/minimum-size-subarray-sum)

## 题目简介

```js
/**
 * @param {number} target
 * @param {number[]} nums
 * @return {number}
 */
```

题目要求我们在 nums 数组中找出长度最小且满足和大于等于 target 的子数组

最后返回其长度

## 解题思路

这道题我们要使用滑动窗口的思路

如果当前窗口内的子数组之和小于 target，则窗口右边界往右移动，窗口扩大

如果当前窗口内子数组之和大于 target，则窗口左边界往右移动，窗口缩小

我们还需要一个变量来记录满足要求的最小窗口大小

最后返回这个变量的值即可

### Javascript

```javascript
/**
 * @param {number} target
 * @param {number[]} nums
 * @return {number}
 */
var minSubArrayLen = function (target, nums) {
    let res = Number.MAX_SAFE_INTEGER
    let left = 0
    let right = 0
    let sum = nums[0]
    while (right < nums.length) {
        if (sum >= target) {
            res = Math.min(res, right - left + 1)
            sum -= nums[left]
            left++
        } else {
            right++
            sum += nums[right]
        }
    }
    return res === Number.MAX_SAFE_INTEGER ? 0 : res
};
```

### 复杂度分析
#### 时间

O(n)

#### 空间

O(1)
