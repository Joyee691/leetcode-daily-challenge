# 53. Maximum Subarray

> [Leetcode link](https://leetcode.com/problems/maximum-subarray)

## 题目简介

```js
/**
 * @param {number[]} nums
 * @return {number}
 */
```

题目给我们一个数字数组 nums

要求我们在数组中找到一个和最大的子数组并返回其和

### Javascript

```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var maxSubArray = function(nums) {
    let sum = 0
    let res = nums[0]

    for(const num of nums) {
        sum = Math.max(num, num + sum)
        res = Math.max(res, sum)
    }

    return res
};
```
