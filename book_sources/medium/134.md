# 134. Gas Station

> [Leetcode link](https://leetcode.com/problems/gas-station)

## 题目简介

```js
/**
 * @param {number[]} gas
 * @param {number[]} cost
 * @return {number}
 */
```

有一个环形道路，它有多个停靠站，每个停靠站可以补充 `gas[i]` 的油量，从当前点跑到下一个停靠站需要使用 `cost[i]` 的油

汽车有无限大的邮箱

题目要求我们选择一个起始点（下标坐标）使得汽车可以有足够的油跑完整个环形道路回到原点

## 解题思路

想要跑完环形道路意味着我们要在每个停靠站加够到下一个停靠站的油，也就是说有个大前提：**所有 gas 提供的油必须大于 cost 消耗的油**

在满足这个大前提的解题步骤如下：

1. 我们选择任意一个点为起始点（为了方便数组遍历我们使用下标 0 作为起始点）
2. 记录跑到下一个停靠点还会剩多少油
3. 如果剩下的油小于 0（意味着跑不到下一个停靠点）我们需要把起始点重新定成下一个点（i+1）
4. 持续遍历直到结束，我们记录的起始点就是题目所需的起始点

当然我们也可以在遍历过程中记录 gas 与 cost 的总和最后比较

### Javascript

```javascript
/**
 * @param {number[]} gas
 * @param {number[]} cost
 * @return {number}
 */
var canCompleteCircuit = function (gas, cost) {
    const totalGas = gas.reduce((acc, cur) => acc + cur, 0)
    const totalCost = cost.reduce((acc, cur) => acc + cur, 0)
    if (totalGas < totalCost) {
        return -1
    }

    let curGas = 0
    let startIdx = 0
    for (let i = 0; i < gas.length; i++) {
        curGas += gas[i] - cost[i]
        if (curGas < 0) {
            curGas = 0
            startIdx = i + 1
        }
    }
    return startIdx
};
```

#### 时间

O(n)

#### 空间

O(1)
